Password recovery

DockCraft runs in your own Docker container — there is no central account service to reset through. To recover access, ask the person who set up the server to:

  1. Open a shell on the Docker host.
  2. Run docker exec -it dockcraft node -e "console.log(require('bcryptjs').hashSync('new-password', 10))".
  3. Paste the resulting hash into dockcraft.config.json under admin.passwordHash and restart the container.
